User Data handling and usage:

For each direct interaction with the bot, I collect and store the name and ID of the user and Discord server. The bot's output (the stats) is also stored.

The way I use this data is for statistical, analytical, and quality purposes, as well as monitoring for any bad content that breaks the Terms of Service, such as harmful usernames or server names. I store basic user and server data for the purposes of being able to block users and servers from using the bot if they are found to be abusing the bot. I store the bot output (stats, image or text format) to continously improve the quality of the bot and to correct potential issues. No identifying information is ever shared with any parties.

User Locale, and how I use it:

This bot has access to the language your Discord client currently is set to. It uses this language to PUBLICLY display stats and messages in your language*. If you don't want the bot to use your client language, you must specify a language yourself with the "language" option in the command.

Deleting your stats image**:

After every stats output, you have 15 seconds to delete it by clicking the red "Delete" button.
Since these images are ONLY hosted on Discord, this will remove the image completely***.
